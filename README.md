# 前端实用资源分享网站（前后端分离）
##主要技术栈：
前端：Vue2全家桶+Ant Design+BootStrap，实现了移动端适配
后端：nodejs+express+mongodb，实现了点赞，文章浏览量等功能

## 启动前台项目

```
npm install
```

## 注意！！！
###本项目为前后端分离项目，后台项目在端口6666启动，只启动前台项目的话没有任何数据。

##启动后台项目

###！！！后台项目我放在了main分支中！！！
###我在app.js中完成了mongodb数据库连接，所以只需在根目录下启动后台服务（app.js）即可

```
node app.js
```
